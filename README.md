# Design Concepts — Parastoo Sadeghi

A set of self-directed product design case studies exploring different problem spaces — mobile utility, B2B SaaS, wellbeing, consumer, and game UX. Each project includes research framing, key design decisions, and high-fidelity screens.

## Case Studies

- **[Transit](https://claude.ai/code/artifact/7c20ab0c-1da4-40b5-9d98-62594b95bbb6)** — real-time transit app redesign, rebuilding trust in ETAs through honest uncertainty instead of false precision.
- **[Huddle](https://claude.ai/code/artifact/f2b0500a-0f4a-4546-af54-e56f9bd46937)** — async video feedback tool for remote design/product teams, built to replace the reflex to schedule a call.
- **[Bloom](https://claude.ai/code/artifact/3b287b16-037f-489c-9ee4-936498f2a75d)** — workplace wellbeing check-in app designed so honesty feels safe, not risky.
- **[Wavelength](https://claude.ai/code/artifact/5cb68133-5d05-47f0-8a29-b3633267bd91)** — mood-first music player built to cut decision fatigue.
- **[Drift Atlas](https://claude.ai/code/artifact/1f8a4049-dfdb-4cf1-abce-216b9ec794a5)** — cozy mobile exploration game UX with no anxiety-driven mechanics.

## About

All five are original concept projects designed to demonstrate range across product, wellness, and B2B design — not affiliated with or built for any existing company.
